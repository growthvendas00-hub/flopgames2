IMAGENS DO CHECKOUT
===================

xbox-card.png  -> Imagem do card Xbox Game Pass Ultimate (PNG TRANSPARENTE).
   Salve aqui com EXATAMENTE este nome: xbox-card.png
   O HTML usa <img class="giftcard-img" src="img/xbox-card.png">. O CSS mantem
   a flutuacao, o reflexo embaixo e o glow verde ao redor (drop-shadow).
   IMPORTANTE: este e o funil de 3 MESES (R$ 40) - de preferencia use um card
   que diga "3 MESES". Se usar .jpg, troque tambem o src no index.html.

bg-portal.jpg  -> Foto de fundo do portal verde.
   Salve aqui a imagem do portal (a que você enviou) com EXATAMENTE este nome:
   bg-portal.jpg

   O CSS (.bg::before em style.css) já aplica máscara + escurecimento para a
   foto se integrar ao tema, sem ficar com cara de "PNG colado". Se o arquivo
   não existir, o fundo usa um gradiente verde de portal como fallback.

   Recomendado: imagem vertical/retrato, no mínimo 1080px de largura, .jpg
   (mais leve). Para o card de R$ 40 (repo flopgames2), salve a mesma imagem
   em Checkout2/public/img/bg-portal.jpg.
